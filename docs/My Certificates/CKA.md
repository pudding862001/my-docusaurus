### **Exam Tips:** 

* 1. Be careful with the time limit, need to be proficient with kubernetes and Linux commands. (e.g.: lscpu, grep, crictl, systemctl)
* 2. Do as more practices as possible. (You can get two free simulator tests from CNCF.)
* 3. Be familiar with networking field, including service, ingress, ingress class, gateway API and HTTPRoute.
* 4. Be familiar with storage topics, including storage class, persistent volume and persistent volume claim.
* 5. Be familiar with autoscaling, helm and troubleshooting.
